## Installation

Browser:
```html
<script src="ovniutils.min.js"></script>
```

Deno:
```js
import * as OU from 'https://esm.sh/ovni-utils@0.2.0'
```

Node:
```sh
$ npm i ovni-utils
```
```js
import * as OU from 'ovni-utils'
```
